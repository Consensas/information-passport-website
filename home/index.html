Consensas Information Passport - Demonstration

Introduction

An Information Passport is a signed digital document that makes some claim, for example "So and so was Vaccinated against COVID-19 on a certain date". If the signature matches the public key of the digital document, the document is Verified. If the Claim made in the document checks against a set of (arbitrary) rules and the "fingerprint" of the public key is known, we say the document is Validated.

A Vaccination Passport is an Information Passport that provides digital proof of a Vaccination. A Test Record Passport is an Information Passport that provides digital proof a some test having been performed.
