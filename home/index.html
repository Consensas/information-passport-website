Information Passport

An **Information Passport** is a _signed_ digital document that makes some claim, for example "So and so was Vaccinated against COVID-19 on a certain date". If the _signature_ matches the _public key_ of the digital document, the document is _verified_. If the _claim_ made in the document checks against a set of (arbitrary) rules _and_ the "fingerprint" of the public key is known, we say the document is _validated_.
